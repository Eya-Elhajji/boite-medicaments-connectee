# Boîte à médicaments connectée
Projet professionnel réalisé à l'AJST.

## Description
Boîte à médicaments intelligente rappelant les prises et détectant automatiquement si la dose du jour a été retirée. Le capteur infrarouge FC-51 identifie la prise, un buzzer déclenche une alarme en cas d’oubli, et l’écran LCD I2C affiche les informations en temps réel.

## Matériel utilisé
- Arduino Nano
- Capteur infrarouge FC-51
- Écran LCD 16x2 I2C
- Buzzer

## Fonctionnement
1. Le capteur détecte si la dose a été prise.  
2. Le buzzer sonne automatiquement en cas d’oubli.  
3. L’écran LCD affiche l’état en temps réel pour faciliter le suivi.
