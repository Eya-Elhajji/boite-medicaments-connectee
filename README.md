# boite-medicaments-connectee
Boîte à médicaments connectée rappelant les prises et détectant automatiquement si la dose du jour a été retirée. Le capteur FC-51 vérifie la prise, un buzzer alerte en cas d’oubli et un écran LCD I2C affiche les informations en temps réel. Projet basé sur Arduino Nano, incluant programmation, câblage et tests.
